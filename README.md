# My old personal website

  
# Prerequisites
  - Node Modules
  - Vue.js

# Project Setup
First step, we need to clone backend api repo to your local machine.

     $ git clone git@github.com:mertcaliskan01/vue-my-old-website.git
     
# Run Project
     $ vue ui
     $ Tasks -> Serve -> Run Task

# Tech
  - vue: 2.6.11
  - axios: 0.21.1
  - vuex: 3.4.0
  - vuetify: 2.6.3
  - vue-router: 3.2.0
  - express: 4.17.1
